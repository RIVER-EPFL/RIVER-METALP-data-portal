Filter the inventory by 'Parameter category'.

The 'param_category', 'column_calculated' and 'calcul_func' fields can be maximum 45 characters long.

In the 'columns_used' field, all columns needed for the calculation must be listed in a comma-separated format (i.e. col1,col2,col3).

All fields are mandatory.

The rows can be reordered by dragging and droppping them at their new location. The draggable zone is the first column. The 'Update Order' button must be pressed to save the changes. Beware that to have the excpeted result, the table must be ordered in ascending order using the 'order' column and must not be filtered.
