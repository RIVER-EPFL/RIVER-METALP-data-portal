The option_name, data, sd and min_max columns can be 75 characters long maximum.

The data, sd and min_max fields can have multiple values by separating them with a comma (i.e., 'value1,value2,value3').

The param_name can be 25 characters long maximum.

The units can be 10 characters long maximum.

The section_name, option_name, param_name, units, data and plot_func fields are mandatory.

The possible plotting functions are the following:

  • basicPlot:a simple scatter plot with a Loess fitting curve.
  • multiPlot:same as the basicPlot but with multiple parameters.
  • sdPlot:same as the basicPlot but with error bars indicating the standard deviation.
  • minMaxPlot:same as the basicPlot with minimum and maximum values displayed for each measurement.

Fields definition:

  • section_name:defines the parameter groups displayed in the dropdown menu.
  • option_name:defines the parameter name displayed in the dropdown menu.
  • param_name:the parameter name or acronym displayed in the Y-axis of the plots.
  • units:the units of the parameters, displayed in the Y-axis of the plots.
  • data:the name of the parameter in the SQL table (it has to exactly match to allow the parameter to be displayed correctly).
  • sd:the parameters in the SQL table that refers for a standard deviation of a parameter (mainly used for sdPlot plot).
  • min_max:the parameters in the SQL table that refers to a minimum or a maxim of a parameter (mainly used for minMaxPlot plot).
  • plot_func:defines the type of plot function that will be displayed.
