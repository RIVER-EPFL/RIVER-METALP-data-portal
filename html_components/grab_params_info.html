The option name, data, sd and min_max columns can be 75 characters long maximum.

The param name can be 25 characters long maximum.

The units can be 10 characters long maximum.

Section name, option name, param name, units, data and plot func are mandatory.

The possible plotting functions are the following:

  • basicPlot:A simple scatter plot with a Loess fitting curve.
  • multiPlot:The same as the basicPlot but with multiple parameters.
  • sdPlot:The same as the basicPlot but with error bars usind the standard deviation.
  • minMaxPlot:The same as the basicPlot but with the minimum and maximum values displayed for each measurment.
