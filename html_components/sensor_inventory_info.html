Use the 'Station', 'Parameter' inputs to filter the table.

The station name must be an acronyme of exactly 3 characters long.

The param_name and serial_nb fields can be 25 characters long maximum.

The param_full and model fields can be 75 characters long maximum.

The calibration field is created automatically when both the calibration_a and calibration_b fields are filled.

The param_name, param_full and model fields are mandatory.
