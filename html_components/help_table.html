Date Range: You can filter the data using a date. The minimum filter resolution is “daily”. You can reset the date to the maximum available range with the reset button. The date range is shared by all the displayed plots.
Hide/show input button: To hide or show the sidebar containing the plot input elements to increase the plot dimensions.
Add unit button: To add a new visualisation unit to the page.
Remove unit button: To remove the last added visualisation unit from the page.
Sidebar inputs: Select any inputs of the sidebar to filter and display specific data of our monitoring network.
Plot interactions: Time serie plots are brushable, meaning that you can select the date range by clicking and horizontally dragging the mouse on the plot. The maximum filtering resolution is “daily”.
By double clicking the plot will reset the date range to the maximum range available. These actions will affect the date range of all the displayed plot.
By hovering the mouse on a data point, data information will be displayed.
