The option_name, param_name, data and grab_param_name columns can be 75 characters long maximum.

The grab_param_name field can have multiple values by separating them with a comma (i.e., 'value1,value2,value3').

The units can be 10 characters long maximum.

The section_name, option_name, param_name, units and data fields are mandatory.

Fields definition:

  • section_name:defines the parameter groups displayed in the dropdown menu.
  • option_name:defines the parameter name displayed in the dropdown menu.
  • param_name:the parameter name or acronym displayed in the Y-axis of the plots.
  • units:the units of the parameters, displayed in the Y-axis of the plots.
  • data:the name of the sensor parameter from csv file generated with the SensorDataPreprocessing tool (it has to exactly match to allow the parameter to be displayed correctly).
  • grab_param_name:the name of the grab sample parameter in the SQL table (it has to exactly match to allow the parameter to be displayed correctly).
