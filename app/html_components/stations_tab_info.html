The 'station' name must be an acronyme of exactly 3 characters long.

The 'full_name' can have a maximum of 24 characters.

The 'color' must be in a hexadecimal format (i.e. #3d84b0), hence exactly 7 characters long.

The 'name', 'full_name', 'catchment' and 'color' fields are mandatory.

The NULL keyword must be entered to explicitly delete a non-mandatory value.

The rows can be resorted by dragging and dropping them at their new location. The draggable zone is the first column. The 'Update Order' button must be pressed to save the changes. Beware that to have the expected result, the table must be sorted in ascending order using the 'order' column and must not be filtered.
